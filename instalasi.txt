Cara install aplikasi keepkas di localhost atau xampp :

1. Buat folder baru dengan nama 'kasku' didalam "Program C/xampp/htdocs".
2. Ekstrak file zip kasku ke dalam folder tersebut.
3. Buka aplikasi XAMMP, kemudian tekan "Start" pada Apache dan MySQL.
4. Buat database dengan nama kasku_mvc dari 'localhost/phpmyadmin'.
5. Kemudian pilih tombol 'Import' yang ada di menu atas.
6. Pilih file database yang ada di folder "kasku/database". 
7. Lalu tekan 'Go' atau 'Kirim'.
8. Selanjutnya anda bisa mengakses aplikasi ini dengan url "localhost/kasku".

==============================
Contoh Akun User (anggota kelas) :
url	= localhost/kasku
nim 	= 01001
password	= 01001

==============================
Akun Admin (bendahara kelas) :
url	= localhost/kasku/admin
username	= admin
password	= admin


Note : File configurasi ada di dalam folder "app/config/config.php". Silahkan sesuaikan dengan nama folder instalasi
dan database anda.
